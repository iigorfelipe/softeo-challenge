## Installation

- first clone and enter project folder

```bash
git clone git@github.com:iigorfelipe/softeo-challenge.git
```

```bash
cd softeo-challenge
```

- then enter the frontend and backend folders to install them

### Frontend

```bash
cd frontend
```

```bash
npm install
```

### backend

```bash
cd backend
```

```bash
npm install
```

## Local Usage

### Frontend

```bash
cd frontend
```

```bash
npm run dev
```

### backend

```bash
cd backend
```

```bash
npm run dev
```
